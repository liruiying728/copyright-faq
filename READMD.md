# 什么是版权？

版权，也称为著作权，是著作权人对其作品所专有的权利。著作权一词通常为法律文件中所使用，而著作权一般被习惯称呼为版权。

版权要保障的是思想的表达形式，而不是保护思想本身，而且其是有期限的权利，在一定期限经过后，著作财产权即归于失效，而属公有领域，任何人皆可自由利用。

著作权又包含著作人格权和著作财产权。

参考来源：[著作权](https://zh.wikipedia.org/zh-cn/%E8%91%97%E4%BD%9C%E6%AC%8A)

拓展阅读：[公有领域介绍](https://zh.wikipedia.org/wiki/%E5%85%AC%E6%9C%89%E9%A2%86%E5%9F%9F)

# 版权保护的是什么？

著作权的客体是作品。只要不违反该国家或地区所参加的国际条约的要求，对作品如何分类、对各类作品施加何种程度的保护依其各自法律规定。在中国，受著作权保护的作品，是指文学、艺术和科学领域内具有独创性并能以某种有形形式复制的智力成果。依在世界范围内有较多签约国的《伯尔尼保护文学和艺术作品公约》第二条第一款的规定，「文学和艺术作品」一词包括文学、科学和艺术领域内的一切成果，不论其表现形式或方式如何，诸如：

- 书籍、小册子和其他文学作品；
- 讲课、演讲、讲道和其他同类性质作品；
- 戏剧或音乐戏剧作品；
- 舞蹈艺术作品和哑剧作品；
- 配词或未配词的乐曲；
- 电影作品和以类似摄制电影的方法表现的作品；
- 图画、油画、建筑、雕塑、雕刻和版画作品；
- 摄影作品和以类似摄影的方法表现的作品；
- 实用艺术作品；
- 与地理、地形、建筑或科学有关的插图、地图、设计图、草图和立体作品。

另外：

- 凡是中国公民、法人或者非法人单位的作品，不论是否发表都享有著作权；
- 外国人的作品首先在中国境内发表的，也依著作权法享有著作权；
- 外国人在中国境外发表的作品，根据其所属国与中国签订的协议或者共同参加的国际条约享有著作权。

参考来源：[著作权](https://zh.wikipedia.org/zh-cn/%E8%91%97%E4%BD%9C%E6%AC%8A)

拓展阅读：[《伯尔尼保护文学和艺术作品公约》介绍](https://zh.wikipedia.org/wiki/%E4%BC%AF%E5%B0%94%E5%B0%BC%E4%BF%9D%E6%8A%A4%E6%96%87%E5%AD%A6%E5%92%8C%E8%89%BA%E6%9C%AF%E4%BD%9C%E5%93%81%E5%85%AC%E7%BA%A6)、[《伯尔尼保护文学和艺术作品公约》全文](http://ipr.mofcom.gov.cn/zhuanti/law/conventions/wipo/1/Berne_Convention.html)

# 如何界定侵权？

笼统地讲，未获得授权、超出或者违反授权范围情况下使⽤非免费资源进行商业活动（不管是否盈利）均属侵权。这里有一个重点——商业使用，也就是说，在个人作品展示中使用付费字体，理论上不侵权。

参考来源：[怎样界定一个字体的版权时候隶属侵权](https://www.17font.com/news/qa-20200526171744.html)

# 常见侵权举例

- 收费字体在未购买版权的情况下⽤于商业⽤途；

### Q：使用 Midjourney、Stable Diffusion 等产品设计的 AI 图片版权归属设计者吗？

## 字体类

### Q：图⽚（海报、banner 等）中⽂字使⽤某款商⽤字体中的⼀个字算侵权吗？

A：算侵权。

字体侵权不以字数多少计算，只要在没有授权下使⽤便是侵权。

相关新闻：[《九层妖塔》字体侵权纠纷 7 个字判赔 14 万元](http://ip.people.com.cn/n1/2018/0604/c179663-30033246.html)、[《失恋 33 天》影⽚字体涉嫌侵权造字⼯房将起诉](http://tech.sina.com.cn/i/2012-04-01/19026906604.shtml)

### Q：修改侵权字体多于三处算侵权吗？

A：可能侵权。

修改字体的多少是否侵权要看修改后的字体是否具备原创性，不能仅从「修改了多少处」去判断。可参考 [正版字体这样改，就能免费无版权](https://www.shejidaren.com/zhengban-ziti-xiugai.html) 的修改方法进行「大变样」从而规避风险。

### Q：在⽹⻚端引⽤第三⽅字体（font-family）侵权吗？

A：不侵权。

不管在 font-family 中定义了多少种商⽤字体都不侵权，因为第三⽅的字体存在于客户端电脑上，并不属于本身⽹⻚端项⽬中的⼀部分；且 CSS 只告知客户端浏览器应该⽤何种字体渲染，字体不包含在项⽬中。故不侵权。

### Q：在⽹⻚端嵌⼊第三⽅字体（@font-face）侵权吗？

A：可能侵权。

字体已被包含⼊项⽬中，客户端电脑没此字体时是从⽹⻚端上下载使⽤。如⽆获得授权商⽤、⾮公开或者不被允许商⽤嵌⼊的话则属侵权。

### Q：App、客户端软件内嵌商⽤字体侵权吗？

A：可能侵权。

原理同「Q：在⽹⻚端嵌⼊第三⽅字体（@font-face）侵权吗？」。

### Q：微软雅⿊是免费字体吗？

A：有限范围内免费。

微软雅⿊体是微软委托⽅正设计的，以商业发布为⽬的的微软雅⿊版权，仍由⽅正保留。

允许情况：

- 在 macOS 或 Windows 系统中，安装字体库使⽤。
- 个⼈打印、微博图⽚、美化照⽚、学⽣作业等所有个⼈⾮商业⽤途。

其他的，基本都属于侵权行为，包括但不仅限于全媒体商业发布，包括印刷媒体和电⼦媒体；商业设计案，如⼿册、海报、平⾯⼴告、户外⼴告、影视⼴告、产品包装等；企业 VI 系统。新媒体，以图⽚的形式在 App 软件、微信公众号等平台宣传使⽤；编辑好的含有未授权字体（如微软雅⿊）⽂件发送⾄客户处、公开展示的等。

参考来源：[微软雅⿊的知识产权](https://baike.baidu.com/item/%E5%BE%AE%E8%BD%AF%E9%9B%85%E2%BF%8A%E4%BD%93/988302#8)

### Q：苹⽅是免费字体吗？

A：有限范围内免费。

苹⽅是 Apple 公司委托华康设计的，苹⽅是 Apple 的商标，华康拥有版权，华康⽅对外授权的「苹⽅」名称是「华康⾦刚⿊」。

允许情况：在合法 macOS 以及相关软件前提下使⽤、显示、打印的各种图⽚、视频都是在合法授权范围，⽆论是商⽤或⾮商⽤，如海报、平⾯设计、户外⼴告、视频图⽚、⼿提袋等。

⽆授权下，不允许情况：将字体取出或搭建在⾮ macOS 的第三⽅软硬件上，如 App、游戏等；在⾮ macOS 上⽆授权使⽤；在「⿊苹果」上使⽤。

### Q：微软雅⿊、苹⽅打印出来成 pdf、纸质⽂件、图⽚侵权吗？

A：有限范围内免费。

微软雅⿊在没有授权情况下，因商业⾏为（包含间接、直接⾏为） 打印出来、转换成 pdf 发送给客户、公开演讲等⾏为是侵权的，个⼈⾮商⽤则被允许。

苹⽅在允许范围内可以免费商⽤、个⼈使⽤（详⻅本文 FAQ 章节的「Q：苹⽅是免费字体吗？」）。

### Q：使⽤ DIN 字体能免费商⽤吗？

A：不能。

DIN、DIN1451、DINNext 等「DIN」家族字体都有版权，需付费才可商⽤。如 DIN1451 属于 Linotype 公司所有，此公司在 2006 年被 Monotype 收购，后续⽅正独家代理了 Monotype ⻄⽂字体产品在中国⼤陆地区的销售，故最后版权归于⽅正所有，⽽⽅正未声明可其可免费商⽤。

参考来源：[DIN 1451](https://en.wikipedia.org/wiki/DIN_1451) 、[DIN 1451 在 myfonts 上的许可说明](https://www.myfonts.com/fonts/linotype/din-1451?tab=licensing)

### Q：华康系列字体是免费商⽤的吗？

A：有限范围内免费商用。

华康系列字体如⽆购买授权，能免费商⽤的只能在阿⾥巴巴集团旗下平台商家使⽤，并且「其它使用需求，包括打印、一般文书等使用目的，须另行购买正式产品」，除此之外其它平台的使⽤都属侵权。

参考来源：[华康字型与阿里巴巴集团旗下平台授权字库说明](https://www.dynacw.com.cn/news/news_detail.aspx?s=7)

### Q：在第三⽅渠道（如淘宝）购买了商⽤字体，使⽤后侵权吗？

A：可能侵权。

购买字体渠道请优先使⽤字体⼚商官⽹或其官⽅商城，了解清楚授权范围等信息。淘宝上的「可商⽤⽆版权纠纷字体」、「正版字体授权购买」有的属于个⼈商家售卖⾏为，可能是⾮正规授权途径，难免出现纠纷，建议不采⽤或慎重使⽤。

常见字库官网：[⽅正](https://www.foundertype.com/)、[造字⼯房](https://www.makefont.com/index.html)、[汉仪](https://www.hanyi.com.cn/)、[文鼎](https://www.arphic.com.cn/)

### Q：⽅正公开的「⽅正⿊体、⽅正书宋、⽅正仿宋、⽅正楷体、方正甲骨文」是免费商⽤的吗？

A：是的，但有风险。

虽 [方正官网](https://www.foundertype.com/index.php/FontInfo/index/id/131) 宣称字体属性是「免费字」，并备注「仅商业发布范围使用，可下载直接使用无需获得书面授权」，但其顶部又规定「在使用字体时需要获取字体授权，具体授权包括：个人非商业授权、商业发布授权……嵌入式应用授权」。前后表述有冲突，故字体使⽤有⻛险，建议不使⽤或慎重使⽤。

### Q：使⽤了第三⽅字体，仅仅加了图层样式算侵权吗？

A：算侵权。

简单变更字体颜⾊、投影、形变等皆属于直接使⽤该字体，⽆商⽤授权情况下属侵权。

### Q：购买的可商⽤字体可以⽤在项⽬、产品、企业 logo 上吗？

A：需要结合实际情况判断。

购买了的字体有⽤途范围说明，因为适⽤范围不同，价格也是不同的。如有明确标明可以运⽤在企业 VI 视觉上的则可使⽤。可参考 [⽅正字体授权分类](https://www.foundertype.com/index.php/FontInfo/buyFontauth/fid/21659.html) 的介绍辅助理解。

### Q：Google Fonts 上的字体是可以随便使⽤吗？

A：是的。

所有字体均在开源许可证下发布，可以在任何⾮商业或商业项⽬中使⽤。

参考来源：[Google Fonts 官⽹说明](https://developers.google.com/fonts)

### Q：截了含系统字体或其它商⽤字体的图印刷在企业物料上，侵权吗？

原因可能是：

- 如果是 Windows 的系统字体（如微软雅⿊），由于⽅正只授权微软、⽤户⽤于电脑显示，其它⽅式的使⽤都涉及侵权。
- 在合法 macOS 下使⽤苹⽅字体，因为在合法授权范围，所以不算侵权。
- 其他商⽤字体可能要看在整张图⽚中的⽐重、作⽤等。如在图⽚中截图表示产品内容的，仅做说明作⽤且占⽐很⼩的则可能不侵权；如截图中的⽂字被放⼤当标题使⽤，则可能涉及侵权。

## ⾳效、⾳乐类

### Q：线上线下商⽤使⽤了⾳乐做 bgm 侵权吗？

A：可能侵权。

除了声明⽆版权的⾳乐外，⼀般⾳乐（歌曲、⾳效、铃声等）商⽤是有⼀定使⽤限定的，如限制使⽤场景、内置应⽤等。使⽤前需获得授权。可参考 [摄图⽹⾳乐商⽤报价表](https://699pic.com/music/vipCompany.html?click_type=671) 的介绍辅助理解。

### Q：购买了⾳乐内置到 app 后，app 能售卖给客户去做⼆次开发吗？

A：不可以。

「售卖给客户去做⼆次开发」有「转售」的概念，转售与内置属于两种不同授权范围，需要明确该⾳乐是否有内置、转售授权后⽅可进⾏下⼀步使⽤。

### Q：购买的⾳乐商⽤除了常规的授权范围外，还需要留意什么？

A：还需留意：

- 是否具有全球授权权限（是否允许除了中国外的国家或地区使⽤）。
- ⾳乐内置到产品后转售给客户，是否需要把客户加入到授权⽅⽩名单中。
- 授权终端是否全平台、语⾔是否仅限中⽂。
- 其他讨论细节是否会最后在合同中体现等。

## 其它

### Q：公司内部活动的物料使⽤第三⽅字体、图⽚侵权吗？

A：可能侵权。

公司内部活动虽也属于商业行为，虽然面向企业内部、传播范围有限，但不排除会因泄露而产生商业风险。

### Q：购买了字体、图⽚，就可以随意使⽤吗？

A：不可以。

每款字体、每张图⽚的授权主体（是否有限定为集团公司企业旗下单⼀⼦品牌、⼦公司或产品线使⽤）、授权时限（⼀年、五年、永久或其它时⻓）、授权发布渠道（线上、线下、内嵌等）都会有不同，⽆法⼀概⽽论，具体需要看授权时的合同约定。

### Q：标明了「图⽂来源于⽹络，如有侵权，请联系删除」是否就万事⼤吉？

A：不是。

未经著作权方的书面授权，就是侵权。有标明这句话恰恰是自证其未核实网络来源的图片是否侵权，类似的还有「图片版权属于原作者，如有侵权，请联系删除」。所以，如果图片是有版权要求的，即会导致侵权。

### Q：公司购买了摄图网的会员，是不是下载的图片和视频全部都能商用？

A：不是。

摄图网的资源下载页右侧，授权范围标注除了有「可做商用」外，还有「非营利使用」、「不可商用」等，建议使用前进行二次确认。

### Q：我帮甲方做的设计图，版权归甲方吗？

A：视具体情况而定。

如果甲方没有和你约定、订立版权归属协议，版权归你，甲方只有使用权。

参考来源：[委托作品的著作权归属于谁？](http://www.ipwq.cn/ipwqnew/show-789.html)
