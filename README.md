## 什么是版权？

版权，也称为著作权，是著作权人对其作品所专有的权利。著作权一词通常为法律文件中所使用，而著作权一般被习惯称呼为版权。

版权要保障的是思想的表达形式，而不是保护思想本身，而且其是有期限的权利，在一定期限经过后，著作财产权即归于失效，而属公有领域，任何人皆可自由利用。

著作权又包含著作人格权和著作财产权。

参考来源：[著作权](https://zh.wikipedia.org/zh-cn/%E8%91%97%E4%BD%9C%E6%AC%8A)

拓展阅读：[公有领域](https://zh.wikipedia.org/wiki/%E5%85%AC%E6%9C%89%E9%A2%86%E5%9F%9F)

## 版权保护的是什么？

著作权的客体是作品。只要不违反该国家或地区所参加的国际条约的要求，对作品如何分类、对各类作品施加何种程度的保护依其各自法律规定。

在中国，受著作权保护的作品，是指文学、艺术和科学领域内具有独创性并能以某种有形形式复制的智力成果。依在世界范围内有较多签约国的《伯尔尼保护文学和艺术作品公约》第二条第一款的规定，「文学和艺术作品」一词包括文学、科学和艺术领域内的一切成果，不论其表现形式或方式如何，诸如：

- 书籍、小册子和其他文学作品
- 讲课、演讲、讲道和其他同类性质作品
- 戏剧或音乐戏剧作品
- 舞蹈艺术作品和哑剧作品
- 配词或未配词的乐曲
- 电影作品和以类似摄制电影的方法表现的作品
- 图画、油画、建筑、雕塑、雕刻和版画作品
- 摄影作品和以类似摄影的方法表现的作品
- 实用艺术作品
- 与地理、地形、建筑或科学有关的插图、地图、设计图、草图和立体作品

另外：

- 凡是中国公民、法人或者非法人单位的作品，不论是否发表都享有著作权
- 外国人的作品首先在中国境内发表的，也依著作权法享有著作权
- 外国人在中国境外发表的作品，根据其所属国与中国签订的协议或者共同参加的国际条约享有著作权

参考来源：[著作权](https://zh.wikipedia.org/zh-cn/%E8%91%97%E4%BD%9C%E6%AC%8A)

拓展阅读：[《伯尔尼保护文学和艺术作品公约》介绍](https://zh.wikipedia.org/wiki/%E4%BC%AF%E5%B0%94%E5%B0%BC%E4%BF%9D%E6%8A%A4%E6%96%87%E5%AD%A6%E5%92%8C%E8%89%BA%E6%9C%AF%E4%BD%9C%E5%93%81%E5%85%AC%E7%BA%A6)、[《伯尔尼保护文学和艺术作品公约》全文](https://ipr.mofcom.gov.cn/zhuanti/law/conventions/wipo/1/Berne_Convention.html)

## 如何界定侵权？

笼统地讲，未获得授权、超出或者违反授权范围使用受版权保护的资源，不论是否用于商业活动，均可能构成侵权。商业使用会让侵权风险显著上升，赔偿额度也更高；但即便是个人非商业场景，也需视具体字体、图片的 EULA（最终用户许可协议）而定，不可简单认为「个人使用就不侵权」。

## 常见侵权举例

字体类

- 收费字体在未购买版权的情况下用于商业用途
- 字体变形较少或无变形（详见 [修改侵权字体多于三处算侵权吗？](#修改侵权字体多于三处算侵权吗)）
- 在 Windows 上使用「苹方」字体
- 编辑好的含有未授权字体（如微软雅黑）文件发送至客户处、公开展示的

图片类

- 涂抹防盗水印
- 多张有版权图片的局部合成一张新的整图，新图最终效果与原图相差不大
- 人物肖像用于敏感行业（如医美、保健、养生、交友等）

其他行为

- 二次售卖有版权文件。

## 版权知识 FAQ

### AI

#### 国内 AI 工具（文心一格、即梦等）生成的图片，版权归谁？

视具体工具的用户协议而定，多数国内工具将版权授予用户，但均附有平台使用该内容改进服务的条款。

以常见工具为例：文心一格（百度）在协议中声明用户拥有 AI 生成内容的著作权；即梦（字节跳动）同样声明用户持有合法生成内容的权益。

建议使用前仔细阅读各工具的用户协议，重点关注「内容归属」和「授权给平台」两个条款。此外，《生成式人工智能服务管理暂行办法》（2023 年 8 月施行）规定，提供者不得将用户依法享有的内容权益归属为平台所有，但具体权利范围仍需结合实际合同条款判断。

参考来源：[《生成式人工智能服务管理暂行办法》全文](https://www.cac.gov.cn/2023-07/13/c_1690898327029107.htm)

#### 用 AI 生成的图片，能用于企业商标注册吗？

目前存在较大风险，不建议直接使用。

中国国家知识产权局（CNIPA）尚未出台明确的 AI 生成内容商标注册指引。若申请人无法证明其对作品具有独立的智力创作贡献，注册申请存在被拒风险。此外，由于 AI 模型可能对多个用户生成近似图像，存在与他人商标构成近似的潜在法律风险。

#### 使用 Midjourney、Stable Diffusion 等产品设计的 AI 图片版权归属设计者吗？

视情况而定，不同工具、不同账户类型、不同国家法律下结论不同。

**Midjourney**：根据其服务条款，免费用户生成的图片版权归 Midjourney 所有；付费订阅用户可主张商业版权，但仍需遵守其服务条款中的限制。

**中国法律视角**：2023 年北京互联网法院在「李昀锴案」中判定，设计者对 AI 辅助生成图片有足够的智力投入（如提示词设计、参数调整等），图片可受著作权法保护。但此为个案判决，立法层面尚未明确统一规定。

**美国法律视角**：美国版权局已多次裁定，纯粹由 AI 生成、无足够人类创作介入的内容不受版权保护。由于 Midjourney 为美国公司，其条款受美国法律约束。

综合来看，AI 生成图片的版权归属，需结合所用工具的账户类型与服务条款、使用者所在地适用的法律、以及设计者对创作过程的实际介入程度三方面判断，不能一概而论。

参考来源：[AI 生成内容受著作权法保护吗？](http://www.iprchn.com/cipnews/news_content.aspx?newsId=138949)、[Terms of Service of Midjourney](https://docs.midjourney.com/docs/terms-of-service)、[《生成式人工智能服务管理暂行办法》全文](https://www.cac.gov.cn/2023-07/13/c_1690898327029107.htm)

### 字体

#### 图片（海报、banner 等）中文字使用某款商用字体中的一个字算侵权吗？

算侵权。

字体侵权不以字数多少计算，只要在没有授权下使用便是侵权。

相关新闻：[《九层妖塔》字体侵权纠纷 7 个字判赔 14 万元](http://ip.people.com.cn/n1/2018/0604/c179663-30033246.html)、[《失恋 33 天》影片字体涉嫌侵权造字工房将起诉](http://tech.sina.com.cn/i/2012-04-01/19026906604.shtml)

#### 修改侵权字体多于三处算侵权吗？

可能侵权。

首先需要澄清：「修改超过三处就不侵权」是业内流传的误解，没有任何法律依据。

无论修改多少处，判断是否侵权的核心标准是：修改后的字体是否已具备独立的原创性，与原字体构成实质性差异，而非单纯以「修改了几处」来计算。可参考 [正版字体这样改，就能免费无版权](https://www.shejidaren.com/zhengban-ziti-xiugai.html) 的修改方法进行「大变样」从而规避风险。

#### 在网页端引用第三方字体（font-family）侵权吗？

不侵权。

不管在 font-family 中定义了多少种商用字体都不侵权，因为第三方的字体存在于客户端电脑上，并不属于本身网页端项目中的一部分；且 CSS 只告知客户端浏览器应该用何种字体渲染，字体不包含在项目中。故不侵权。

#### 在网页端嵌入第三方字体（@font-face）侵权吗？

可能侵权。

字体已被包含入项目中，客户端电脑没此字体时是从网页端上下载使用。如无获得授权商用、非公开或者不被允许商用嵌入的话则属侵权。

#### App、客户端软件内嵌商用字体侵权吗？

可能侵权。

原理同 [在网页端嵌入第三方字体（@font-face）侵权吗？](#在网页端嵌入第三方字体fontface侵权吗)。

#### 微软雅黑是免费字体吗？

有限范围内免费。

微软雅黑体是微软委托方正设计的，以商业发布为目的的微软雅黑版权，仍由方正保留。

允许情况：

- 在正版 Windows 系统中，通过系统预装方式获取并使用该字体
- 个⼈打印、微博图⽚、美化照⽚、学⽣作业等所有个⼈⾮商业⽤途

注意：通过非微软官方渠道另行获取微软雅黑字体文件并安装使用（包括在 macOS 上自行安装），本身可能已属违规获取，不在合法授权范围内。

参考来源：[微软雅黑的知识产权](https://baike.baidu.com/item/%E5%BE%AE%E8%BD%AF%E9%9B%85%E2%BF%8A%E4%BD%93/988302#8)

#### 苹方是免费字体吗？

有限范围内免费。

苹方是 Apple 公司委托华康设计的，苹方是 Apple 的商标，华康拥有版权，华康方对外授权的「苹方」名称是「华康金刚黑」。

允许情况：在合法 macOS 以及相关软件前提下使用、显示、打印的各种图片、视频都是在合法授权范围，无论是商用或非商用，如海报、平面设计、户外广告、视频图片、手提袋等。

无授权下，不允许情况：将字体取出或搭建在非 macOS 的第三方软硬件上，如 App、游戏等；在非 macOS 上无授权使用；在「黑苹果」上使用。

#### 微软雅黑、苹方打印出来成 pdf、纸质文件、图片侵权吗？

有限范围内免费。

微软雅黑在没有授权情况下，因商业行为（包含间接、直接行为） 打印出来、转换成 pdf 发送给客户、公开演讲等行为是侵权的，个人非商用则被允许。

苹方在允许范围内可以免费商用、个人使用（详见 [苹方是免费字体吗？](#苹方是免费字体吗)）。

#### 使用 DIN 字体能免费商用吗？

不能。

DIN、DIN1451、DINNext 等「DIN」家族字体都有版权，需付费才可商⽤。DIN1451 版权归属 Linotype 公司，该公司在 2006 年被 Monotype 收购，版权现归 Monotype 所有。方正为 Monotype 西文字体产品在中国大陆的独家销售代理，在中国大陆购买 DIN 系列字体授权须通过方正渠道，但版权并不归属方正。

参考来源：[DIN 1451](https://en.wikipedia.org/wiki/DIN_1451) 、[DIN 1451 在 myfonts 上的许可说明](https://www.myfonts.com/fonts/linotype/din-1451?tab=licensing)

#### 华康系列字体是免费商用的吗？

有限范围内免费商用。

华康系列字体如无购买授权，能免费商用的只能在阿里巴巴集团旗下平台商家使用，并且「其它使用需求，包括打印、一般文书等使用目的，须另行购买正式产品」，除此之外其它平台的使用都属侵权。

参考来源：[华康字型与阿里巴巴集团旗下平台授权字库说明](https://www.dynacw.com.cn/news/news_detail.aspx?s=7)

#### 在第三方渠道（如淘宝）购买了商用字体，使用后侵权吗？

可能侵权。

购买字体渠道请优先使用字体厂商官网或其官方商城，了解清楚授权范围等信息。淘宝上的「可商用无版权纠纷字体」、「正版字体授权购买」有的属于个人商家售卖行为，可能是非正规授权途径，难免出现纠纷，建议不采用或慎重使用。

常见字库官网：[方正](https://www.foundertype.com/)、[造字工房](https://www.makefont.com/index.html)、[汉仪](https://www.hanyi.com.cn/)、[文鼎](https://www.arphic.com.cn/)

#### 方正公开的「方正黑体、方正书宋、方正仿宋、方正楷体、方正甲骨文」是免费商用的吗？

是的，但有风险。

虽 [方正官网](https://www.foundertype.com/index.php/FontInfo/index/id/131) 宣称字体属性是「免费字」，并备注「仅商业发布范围使用，可下载直接使用无需获得书面授权」，但其顶部又规定「在使用字体时需要获取字体授权，具体授权包括：个人非商业授权、商业发布授权……嵌入式应用授权」。前后表述有冲突，故字体使用有风险，建议不使用或慎重使用。

#### 使用了第三方字体，仅仅加了图层样式算侵权吗？

算侵权。

简单变更字体颜色、投影、形变等皆属于直接使用该字体，无商用授权情况下属侵权。

#### 购买的可商用字体可以用在项目、产品、企业 logo 上吗？

需要结合实际情况判断。

购买了的字体有用途范围说明，因为适用范围不同，价格也是不同的。如有明确标明可以运用在企业 VI 视觉上的则可使用。可参考 [方正字体授权分类](https://www.foundertype.com/index.php/FontInfo/buyFontauth/fid/21659.html) 的介绍辅助理解。

#### Google Fonts 上的字体是可以随便使用吗？

是的。

所有字体均在开源许可证下发布，可以在任何非商业或商业项目中使用。

参考来源：[Google Fonts 官网说明](https://fonts.google.com/)

### 图片

#### 昵图网的图片可以商用吗？

不可以。

昵图网网站声明中规定「关于网站作品均来自网友的上传……由网友自己管理与负责，昵图网不拥有此类作品的版权」、「仅限用于学习交流……勿作它用；若需商业使用，需获得版权拥有者授权」，其版权声明中也规定「用户下载昵图网上的作品，只应在学习、交流、分享的范围内使用，不得用于任何商业用途的范围」。

参考来源：[昵图网的网站声明](https://service.nipic.com/site/wzsm.html)、[昵图网的知识产权声明](http://service.nipic.com/site/copy.html)

#### 公司产品配图中使用了游戏截图侵权吗？

可能侵权。

因下列行为使用的截图应该不侵权：为个人学习、研究或者欣赏，使用他人已经发表的作品的；为介绍、评论某一作品或者说明某一问题，在作品中适当引用他人已经发表的作品。

但游戏的截图有些是有一定的版权限制的，应以对应游戏公司的 EULA（最终用户许可协议）规定为准。如育碧的规定「所有文本、图像、音乐或声音……以及上述内容的任何及所有拷贝均属育碧或其授权商所有」。

参考来源：[游戏截图是否有版权？](https://www.zhihu.com/question/57197968)、[电影的截图和片段截取用于影评，是否侵犯版权？](https://www.zhihu.com/question/38676340)、[育碧终端用户许可协议](https://legal.ubi.com/eula/zh-CN)

#### 截了含系统字体或其它商用字体的图印刷在企业物料上，侵权吗？

可能侵权。

原因可能是：

- 如果是 Windows 的系统字体（如微软雅黑），由于方正只授权微软、用户用于电脑显示，其它方式的使用都涉及侵权
- 在合法 macOS 下使用苹方字体，因为在合法授权范围，所以不算侵权
- 其他商用字体可能要看在整张图片中的比重、作用等。如在图片中截图表示产品内容的，仅做说明作用且占比很小的则可能不侵权；如截图中的文字被放大当标题使用，则可能涉及侵权

#### 可以使用离职员工头像制作商业物料吗？

不可以。

只要行为人没有取得肖像权人的同意，则不得制作、使用、公开肖像权人的肖像，法律另有规定的除外，与是否以营利为目的无关。

参考来源：[离职后，公司还能使用我的照片吗？](https://www.12377.cn/jsal/2021/1afd9177_web.html)

### 音频

#### 线上线下商用使用了音乐做 bgm 侵权吗？

可能侵权。

除了声明无版权的音乐外，一般音乐（歌曲、音效、铃声等）商用是有一定使用限定的，如限制使用场景、内置应用等。使用前需获得授权。可参考 [摄图网音乐商用报价表](https://699pic.com/music/vipCompany.html?click_type=671) 的介绍辅助理解。

#### 购买了音乐内置到 app 后，app 能售卖给客户去做二次开发吗？

不可以。

「售卖给客户去做二次开发」有「转售」的概念，转售与内置属于两种不同授权范围，需要明确该音乐是否有内置、转售授权后方可进行下一步使用。

#### 购买的音乐商用除了常规的授权范围外，还需要留意什么？

还需留意：

- 是否具有全球授权权限（是否允许除了中国外的国家或地区使用）
- 音乐内置到产品后转售给客户，是否需要把客户加入到授权方白名单中
- 授权终端是否全平台、语言是否仅限中文
- 其他讨论细节是否会最后在合同中体现等

### 其他

#### 截图他人作品用于评测、教程或说明，算侵权吗？

不一定，属于著作权法规定的合理使用情形之一。

根据《著作权法》第二十二条，以下情形可以不经著作权人许可、不支付报酬使用已发表作品：为个人学习、研究或欣赏；为介绍、评论某一作品或者说明某一问题，在作品中适当引用。

但需注意「适当」二字——引用的内容不能喧宾夺主，也不能构成对原作品的替代。此外，合理使用通常不适用于商业性宣传目的，公司商业物料中使用截图需另行评估。

参考来源：[《中华人民共和国著作权法》第二十二条](https://www.nppa.gov.cn/xxgk/fdzdgknr/zcfg_210/fl_211/201604/t20160429_4556.html)

#### 标注了 CC 协议的内容可以随意使用吗？

不一定，需要看具体是哪种 CC 协议。

CC（Creative Commons，知识共享）协议有多种类型，授权范围差异很大：

| 协议        | 可商用 | 可修改 | 主要要求                          |
| ----------- | :----: | :----: | --------------------------------- |
| CC0         |   ✅    |   ✅    | 无限制，相当于放入公有领域        |
| CC BY       |   ✅    |   ✅    | 须署名原作者                      |
| CC BY-SA    |   ✅    |   ✅    | 须署名 + 衍生作品须以相同协议发布 |
| CC BY-ND    |   ✅    |   ❌    | 须署名，不可修改                  |
| CC BY-NC    |   ❌    |   ✅    | 须署名，仅限非商业使用            |
| CC BY-NC-SA |   ❌    |   ✅    | 须署名 + 非商业 + 相同协议        |
| CC BY-NC-ND |   ❌    |   ❌    | 须署名 + 非商业 + 不可修改        |

常见误区是看到「CC」就认为可以免费商用。实际上带有「NC」（NonCommercial）的协议明确禁止商业使用。建议使用前仔细核对具体协议类型。

参考来源：[Creative Commons 协议说明](https://creativecommons.org/licenses/?lang=zh)

#### 短视频、直播中使用版权音乐作为 BGM 侵权吗？

可能侵权，各平台规则不同，需分情况处理。

**短视频平台（抖音、快手等）**：平台已与版权方合作，部分歌曲在平台内使用通常不侵权，但将含版权音乐的视频用于站外商业推广，或在未授权歌曲下发布商业视频，则可能被版权方追究。

**B 站**：对版权音乐审核相对严格，商业视频使用未授权版权音乐可能导致视频被下架或收益被转移给版权方。

**直播间**：实时播放版权音乐属于「公开表演权」范畴，即便在平台上也不一定覆盖授权。直播使用背景音乐建议使用平台已授权曲库，或选择无版权 / CC0 音乐。

商业活动线下播放（如门店 BGM、展会现场）同样需要获得授权，通常由中国音乐著作权协会（MCSC）负责授权管理。

#### 用他人照片或截图做成表情包，侵权吗？

可能侵权，且风险来自多个维度。

- **肖像权**：未经本人同意，将他人（尤其是真实可辨认的人物）面部图像制作成表情包，涉及侵犯肖像权。若用于商业推广，风险更高
- **著作权**：截取影视剧画面、游戏截图或插画等制作表情包，可能侵犯原作品的著作权
- **名誉权**：将他人形象配以贬低、侮辱性文字，还可能构成名誉侵权

个人私下发送给朋友的表情包通常处于法律模糊地带；但在企业公众号、官方账号等商业传播场景使用含他人肖像或版权内容的表情包，风险显著上升，建议避免。

#### 使用开源代码库，需要注意版权吗？

需要。开源 ≠ 无版权，开源代码受其 License（许可证）约束，不同 License 的限制差异很大。

| License    | 可商用 | 主要限制                            |
| ---------- | :----: | ----------------------------------- |
| MIT        |   ✅    | 须保留版权声明                      |
| Apache 2.0 |   ✅    | 须保留版权声明，须说明修改内容      |
| GPL v3     |   ✅    | 衍生作品须以 GPL 开源（具有传染性） |
| AGPL       |   ✅    | 网络服务部署也触发开源义务          |
| BSL / SSPL |   ⚠️    | 禁止将软件直接作为服务商业化        |
| 无 License |   ❌    | 不等于公有领域，默认保留所有权利    |

对于设计师来说，常见场景是前端项目引用 npm 包、使用开源 UI 框架（如 Ant Design、Tailwind），这类通常为 MIT 或 Apache 许可，相对宽松。但若产品中用到 GPL 许可的组件，需特别留意其「传染性」条款。

参考来源：[Choose a License](https://choosealicense.com/)

#### 公司内部活动的物料使用第三方字体、图片侵权吗？

可能侵权。

公司内部活动虽也属于商业行为，虽然面向企业内部、传播范围有限，但不排除会因泄露而产生商业风险。

#### 购买了字体、图片，就可以随意使用吗？

不可以。

每款字体、每张图片的授权主体（是否有限定为集团公司企业旗下单一子品牌、子公司或产品线使用）、授权时限（一年、五年、永久或其它时长）、授权发布渠道（线上、线下、内嵌等）都会有不同，无法一概而论，具体需要看授权时的合同约定。

#### 标明了「图文来源于网络，如有侵权，请联系删除」是否就万事大吉？

不是。

未经著作权方的书面授权，就是侵权。有标明这句话恰恰是自证其未核实网络来源的图片是否侵权，类似的还有「图片版权属于原作者，如有侵权，请联系删除」。所以，如果图片是有版权要求的，即会导致侵权。

#### 公司购买了摄图网的会员，是不是下载的图片和视频全部都能商用？

不是。

摄图网的资源下载页右侧，授权范围标注除了有「可做商用」外，还有「非营利使用」、「不可商用」等，建议使用前进行二次确认。

#### 我帮甲方做的设计图，版权归甲方吗？

视具体情况而定，需区分两种情形。

**独立设计师 / 外包乙方**：如果甲方没有和你约定、订立版权归属协议，版权归你，甲方只有使用权。

**甲方公司正式员工**：若设计作品属于职务作品（即在本职工作范围内完成，使用了单位的资源），根据《著作权法》规定，版权通常归单位（雇主）所有，设计师享有署名权。具体以劳动合同和公司规定为准。

无论哪种情况，建议合作前就版权归属、使用范围等在合同中明确约定，避免事后纠纷。

参考来源：[委托作品的著作权归属于谁？](http://www.ipwq.cn/ipwqnew/show-789.html)
